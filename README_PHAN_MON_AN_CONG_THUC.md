# Source phan mon an va cong thuc

Thu muc nay duoc tach tu repo `Tindaptrai/cook-and-order`.

## Da gom

- Controller trang mon an: `DACS_Food/Controllers/MenuController.cs`
- Controller trang cong thuc: `DACS_Food/Controllers/RecipeController.cs`
- API mon an/cong thuc: `DACS_Food/Controllers/Api/FoodsApiController.cs`, `DACS_Food/Controllers/Api/RecipesApiController.cs`
- Model lien quan: `FoodItem.cs`, `FoodCategory.cs`, `Recipe.cs`
- Service va interface: `FoodService.cs`, `Interfaces.cs`
- ViewModel: `MenuViewModel.cs`
- Views: `DACS_Food/Views/Menu`, `DACS_Food/Views/Recipe`, `Views/Home/Menu.cshtml`, `Views/Home/Recipes.cshtml`
- Admin mon an: `Areas/Admin/Controllers/FoodItemsController.cs`, `Areas/Admin/Views/FoodItems`
- Frontend CSS/JS: `menu.css`, `recipes.css`, `menu.js`, `recipes.js`
- Hinh anh mon an/cong thuc: `wwwroot/assets/images/menu`, `wwwroot/assets/images/recipes`, `wwwroot/images/foods`

## Ghi chu

Repo hien tai khong co file am thanh `.mp3`, `.wav`, `.ogg`, `.m4a`, `.aac`.
Neu y ban la "source" thi cac file can cho phan mon an va cong thuc da nam trong thu muc nay.
